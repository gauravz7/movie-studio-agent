# Clarifying questions (ask for a vague idea; skip any the user already answered)

Ask a tight batch (3–5), not an interrogation:

1. **Premise** — what happens, in one or two sentences? (beginning → turn → end)
2. **Tone / genre** — e.g. whimsical, epic, spooky-but-friendly, romantic, comedic.
3. **Length** — how many scenes/frames (default 6–10)?
4. **Audience** — kids, all-ages, adult?
5. **Visual style — ALWAYS ask this explicitly, with options:**
   **photorealistic / cinematic live-action**, **2D cartoon**, **storybook illustration**,
   **3D animation (Pixar-style)**, **anime**, **watercolour**, **claymation**, or their own idea.
   The chosen style becomes the global **style reference** that keeps every scene consistent.
6. **Must-haves** — specific characters, setting, or beats to include?

Then infer sensible defaults for anything unanswered and state them in the treatment for approval.

> **Note on photorealism:** realistic depictions of *people* are more likely to be refused by
> the image filter. If the user picks photorealistic with human characters, warn them and offer
> an illustrated/stylized alternative (or use clearly-adult, original characters) for reliability.
