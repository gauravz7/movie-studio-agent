# Parsing an uploaded script / story

When the user provides a full script, treatment, or story text, extract:

1. **Logline** — one sentence capturing the whole story.
2. **Characters** — every recurring character: name → a one-line *visual* description (age band,
   look, wardrobe) you can turn into a reference sheet. If the script names a copyrighted/IP
   character, propose an **original renamed** equivalent for generation.
3. **Scene breakdown** — an ordered list of scenes, each: location, characters present, and the
   key action/beat. Split long passages into discrete shots.
4. **Style cues** — any tone/era/visual hints → fold into the style guide.

Summarize all four back to the user and ⏸ ask them to confirm or correct **before** building.
Keep the user's own wording where possible; only fill gaps with clearly-labelled assumptions.
